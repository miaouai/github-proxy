👆 点击访问 GitHub
  • GitHub 主站代理 (github.com)
  • Raw 文件代理 (raw.githubusercontent.com)
  • Release 下载代理 (objects.githubusercontent.com)
  • Gist 代理 (gist.githubusercontent.com)
  • 头像代理 (avatars.githubusercontent.com)
  • API 调用支持 (api.github.com)
  • 代码克隆支持 (codeload.github.com)

📋 使用示例

# 访问 GitHub（点击上面 GitHub 图标或访问 /gh）
https://your-domain.com/gh

# 访问仓库
https://your-domain.com/gh/owner/repo

# Clone 仓库
git clone https://your-domain.com/gh/owner/repo.git

# 访问 Raw 文件
https://your-domain.com/gh/raw.githubusercontent.com/owner/repo/main/file.txt

# 直接访问 GitHub 相关域名（自动代理）
https://raw.githubusercontent.com/owner/repo/main/file.txt
